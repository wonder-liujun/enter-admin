
## 简介
本项目采用 Vue3.0、Vite、 Ant-Design-Vue3、TypeScript 等新技术方案，包括二次封装组件、utils、hooks、动态菜单、权限校验、按钮级别权限控制等功能。

## 安装与使用

 > 环境要求: 版本要求Node 14.18+ / 16+ 版本以上，不再支持 Node 12 / 13 / 15。
 > 建议使用pnpm，如果使用yarn,请用Yarn1.x版本，否则依赖可能安装不上。
 
pnpm install
```
- 配置接口地址 `.env.development`
- run

```bash
pnpm serve
```

- build

```bash
pnpm build
```
- 编译项目

```bash
pnpm install

pnpm build
